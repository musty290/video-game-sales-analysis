# 📊 Video Game Sales EDA & Market Insights

## 🚀 Project Overview
This project performs a comprehensive Exploratory Data Analysis (EDA) on global video game sales data to uncover trends, patterns, and actionable business insights.

## 🎯 Business Problem
Which genres, publishers, and platforms drive the most revenue, and how should a gaming company allocate its resources?

## 🧠 Key Analysis Performed
- Data Cleaning & Preprocessing
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis
- Feature Engineering
- Trend Analysis
- Correlation Analysis

## 📈 Key Insights
- Top genres dominate majority of revenue
- Sales are highly skewed
- Strong regional correlations exist
- Market trends show shifts over time

## 💡 Recommendations
- Invest in top-performing genres
- Focus on high-performing publishers
- Apply region-specific strategies
- Reduce exposure to declining segments

- Python (Pandas, NumPy)
- Matplotlib & Seaborn
